# automatic-pet-feeder-arduino

## Project Overview
This projevct is an automated pet feeding system built using arduino
The goal of the system is to dispense food automatically using a servo motor controlled by an arduino board.
The project was developed as part of a Logic Design course and was implemented in a team from concept to final prototype.

This project simulates a basic automated feeding solution for pets when the owner is not available.

## Components Used
- Arduino board
- Servo motor
- breadboard
- jumper wires
- feeding container

## How It Works
The Arduino controls a servo motor that rotates a feeding mechanism.  
When the servo rotates, a portion of food is released from the container.  
The rotation of the servo motor is controlled through Arduino code.
There is also a sensor that tracks the distance from the dispenser to the bowl in order to not dispense food when the bowl is full.
